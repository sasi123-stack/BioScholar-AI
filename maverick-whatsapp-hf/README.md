# Maverick WhatsApp Bot (Llama 4)

Unified Research Engine with active knowledge memory on WhatsApp.

## Setup
1. Set `GROQ_API_KEY` in Settings > Secrets.
2. Set Twilio Webhook to `https://YOUR_SPACE_URL/whatsapp`.
